# Interpreting Binary Data

Data types found in most of the source material are for a 16-bit Intel CPU

Values are `Little Endian`

| Type | Size | Signed |
|------|------|--------|
| byte | 8 bits | false
| char | 8 bits | false
| short | 16 bits | true
| word | 16 bits | false
| unsigned | 16 bits | false
| long | 32 bits | true
| int | 32 bits | true

# Booleans
0 is `false`
Anything else is `true`
